# shecduler
# 🎓 Smart School Scheduler

A modern web-based School Timetable Management System built using **Flask**, **Bootstrap**, and **SQLite** that helps schools manage teachers, classes, subjects, and automatically generate collision-free timetables.

---

## 🚀 FEATURE

### 📊 Dashboard
- Beautiful admin dashboard
- School settings management
- Quick access to major modules

### 👨‍🏫 Teacher Management
- Add new teachers
- View all teachers
- Delete teachers
- Manage teacher information

### 🏫 Class Management
- Add classes and sections
- View available classes
- Delete classes

### 📚 Subject Management
- Add subjects
- Assign weekly periods
- Manage subject information

### 🔗 Class Subject Mapping
- Assign subjects to classes
- Manage class-wise subject allocation

### 🗓 Timetable Generation
- Automatic timetable generation
- Collision-free scheduling
- Teacher conflict prevention
- Class conflict prevention

### 👨‍🏫 Teacher Timetable
- View teacher-wise timetable
- Check lecture schedules

### 🏫 Class Timetable
- View class-wise timetable
- Organized lecture schedule

### ⚙ School Settings
Configure:
- School Start Time
- School End Time
- Lecture Duration
- Lunch Duration
- Lunch After Specific Period
- Working Days
